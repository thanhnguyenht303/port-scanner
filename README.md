# Python TCP Port Scanner

A small multithreaded TCP port scanning project built in Python.

## Features
- Scan a host over a custom port range
- Concurrent scanning with thread pool
- Common TCP service name lookup
- Clean CLI interface
- JSON export for scan results
- Simple unit tests
